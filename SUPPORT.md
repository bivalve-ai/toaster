# Support

Use GitHub Issues for bugs, questions, and feature requests.

Please do not paste raw private session files into public issues. Agent sessions may contain source code, prompts, tool outputs, local paths, secrets, model responses, and private context.

If a reproduction needs session data, prefer a small synthetic fixture that preserves the native shape without private content. If you must share a real trace, redact it first and review the result manually.

For security issues or sensitive disclosures, see [SECURITY.md](./SECURITY.md).
